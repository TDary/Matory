#### Phase 2: High
- [x] **2.1 SceneManager API** - `Transform.FindObjectsOfType<Transform>()` 改为 `SceneManager.GetSceneAt(i).GetRootGameObjects()`

#### Phase 3: Medium
- [x] **3.1 Regex 静态化** - `SecWebSocketKeyRegex` 改为 `static readonly` + `RegexOptions.Compiled`
- [x] **3.2 消除重复 RaycastAll** - `TouchPhase.Moved` 中的第二次 RaycastAll 调用已删除
- [x] **3.3 RaycastResult List 复用** - 添加 `s_raycastResults` 静态字段，所有 case 使用 `Clear()` 复用
- [x] **3.4 移除 LINQ** - `hitInfos.All()` 改为 `AllCollidersAreTriggers()` for 循环；`vecDic.Select...Distinct().OrderBy().ToList()` 改为手动遍历 + Sort
- [x] **3.5 StringBuilder 实例化** - `static StringBuilder` 改为实例字段

#### Phase 4: Bug Fix
- [x] **4.1 GetChildButton 递归修复** - 两个重载都捕获了返回值：`Button result = GetChildButton(child); if (result != null) return result;`
